1.1 QGIS

Quantum GIS is a very well known open source GIS that is available for MS Windows, Linux and MAC-OS.

Please proceed at installing it HERE if necessary. It should be noted that QGIS versions change often, due to the lively developer community that keeps on improving existing modules and adding new ones. In this manual we try to use well-established modules, but it might happen that new QGIS versions change the interface or module availability. In this case please contact the author so that a new version of the manual can be created.

For those savy in programming, QGIS is developed in C++, and supports Python-based QGIS plugin development.
